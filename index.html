Tabulated Data

Tabulated Data is a tool to display data in a tabular format.

Data is passed within the fragment identifier of a URL (the bit after the hash). This must include both the configuration for the table and the values to display. Updating the fragment identifier will update the values displayed in the table.

How to Use

Data must be formatted as stringified JSON which can be parsed with JSON.parse(). The JSON must be structured as follows:

{
    tableConfig: {
        // table configuration
    },
    data: {
        // data
    }
}
